# sanakirja-api
Unofficial API for sanakirja.org


### Instructions
This pakage requires [NodeJS](https://nodejs.org/en/) to run,

1. install the dependencies:
```
$ npm install
```

2. run the tests
```
$ node test.js
```

3. translate multiple words and export the results to './translate.csv' file:
```
$ node translate.js
```
